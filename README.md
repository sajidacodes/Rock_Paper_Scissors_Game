## Description
A simple Python game where you play Rock, Paper, Scissors against the computer.

## Rules
- Rock beats Scissors  
- Scissors beat Paper  
- Paper beats Rock  

## How to Play
1. Run the code.
2. Type your choice (*rock*, *paper*, or *scissors*).
3. The computer chooses too.
4. The program shows who wins or if it's a tie.
